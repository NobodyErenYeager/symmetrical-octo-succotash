6. PgAdmin is asking me for a password but I don't know what it is, what do I do?

Make sure to have installed PostgreSQL and PgAdmin exactly as shown in the video lecture, it should have prompted you for a password. If for some reason it did not, try using the default values, with postgres username and leaving password blank:

username: postgres

password: 

If that does not work for you, try uninstalling PostgreSQL and PgAdmin (make sure to do a full uninstall) and reinstall the software exactly as shown in the video.

You can also try following the steps outlined here.
